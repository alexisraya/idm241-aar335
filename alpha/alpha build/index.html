Walmart Microinteraction - Alpha Build

Interaction: Adding an item into the cart

Notes

Triggers

  • Clicking on the “+ Add” button
  • Hovering on the “+ Add” button
  • Hovering off the “+ Add” button

Rules

  • the pill surrounding the add button increases its length by 50px
  • the words “+ Add” changes to “- 1 +”
  • the item is stored in a data base for the cart
  • (On Hover) pointer changes to cursor
  • (Off Hover) pointer changes to default

Feedback

  • the pill surrounding the add button increases its length by 50px
  • the words “+ Add” changes to “- 1 +”
  • (On Hover) pointer changes to cursor
  • (Off Hover) pointer changes to default

Loops and Modes

  • no loops
  • cart total increases by 1, when the item is added into the cart, the button’s text changes from “+ Add” changes to “- 1 +”

*This button has an additional feature where you can click on the - and + sign to decrease or increase the number in the button*
